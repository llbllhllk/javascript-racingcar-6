## 📄 기능 목록

- 입력

  - [ ] 자동차 이름을 입력받는 기능
  - [ ] 시도 횟수를 입력받는 기능

- 출력

  - [ ] 실행 결과를 출력하는 기능
  - [ ] 최종 우승자를 출력하는 기능

- 기능

  - [ ] 각 자동차의 전진 상태를 반환하는 기능
  - [ ] 최종 우승자를 반환하는 기능

## 🎯 예외 상황

- 경주할 자동차의 이름

  - [ ] 아무것도 입력하지 않았을 경우
  - [ ] 자동차 이름이 중복되었을 경우
  - [ ] 자동차 이름이 5자를 넘었을 경우
  - [ ] 쉼표 구분을 잘못했을 경우

- 시도 횟수

  - [ ] 아무것도 입력하지 않았을 경우
  - [ ] 숫자가 아닌 경우
  - [ ] 음수인 경우
